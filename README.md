Zokugun Folding
===============

**Experimental**

Customize you folding!

## Configuration

In your Settings
```
"folding": {
    "*": {
        "start": "\\{\\{\\{",
        "end": "\\}\\}\\}"
    },
    "typescript": {
        "start": "#region",
        "end": "#endregion"
    }
}
```

## Known Issues

- It's unable to remove existing folding

**Enjoy!**
